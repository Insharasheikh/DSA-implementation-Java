# DSA Implementation Repository 

This repository contains my implementations of Data Structures and Algorithms written from scratch to build a strong understanding of core concepts.

The focus is on clarity, learning, and hands-on practice rather than optimized or production-level solutions.

This repository is shared to help other learners understand the implementation of core data structures from scratch before moving on to problem-solving and logic building.

## Topics Covered

* Arrays
* Linked List
* Stack
* Queue
* Hashing
* Sorting Algorithms
* Trees
* Graph Algorithms

## Goal

To strengthen problem-solving skills and build a solid foundation for coding.
